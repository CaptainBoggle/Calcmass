# Mass
A python script to find the molecular mass in g/mol of a chemical equation
## Example
```
$ molmass C6H12O6
180.15588
$ molmass Cr2O7
215.98800
$ molmass C11H22
154.29238
```
## Setup
You can download using PyPi

Using pip:
```
$ pip install molmass
```
Or download the zip file at https://pypi.python.org/pypi/molmass/0.1.1

Then run:
```
$ python install molmass
```
